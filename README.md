# Paradox-Like-Map-Creator-Online
This is a Map Creator inspired in Paradox's grand strategy games. 
You can:
-Paint and Edit Provinces shapes.
-Create and Merge Countries.
-Export the whole Image to .PNG.
-Download and load the Backup of your Map.
-Customize the transparency between Countries and Reliefs.

I'm interested in adding:
-Timeline made by Historical Maps.
-The Ability to place reference Images to create accurate frontiers.
-Multiplayer Gamemode to play with Friends or other People, just like PX games.

This is Unfinished, and I'm more out than in when it comes to this project, but I'll work in this for now.
